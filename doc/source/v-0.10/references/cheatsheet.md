## Special Template Syntax

* `<foo x="abc">`
* `{% raw %}<foo x="{{expr}}">{% endraw %}`
* `<foo style="name1: value1; name2: value2">`
* `{% raw %}<foo style="name1: value1; name2: {{expr}}; name3: ...">{% endraw %}`
* `<foo class="a b c">`
* `{% raw %}<foo class="a {{expr}} c">{% endraw %}`
* `<foo onclick="methodName">`
* `<foo id="abc">`
* `<foo if="expr">`
* `<foo repeat="item in list">`
* `<foo repeat="(key,item) in list">`
* `<component type="foo">`
* `{% raw %}<component type="{{expr}}">{% endraw %}`

## ViewModel Options

* `data`
* `methods`
* `computed`
* `init`, `created`, `ready`
* `events`

example:

```javascript
module.exports = {

  data: function () {
    return {
      x: 1,
      y: 2
    }
  }

  methods: {
    foo: function () {
      console.log('foo')
    }
  },

  computed: {
    z: function () {
      return this.x + this.y
    }
  },

  events: {
    custom: function (e) {
      console.log(e)
    }
  },

  init: function () {},
  created: function () {},
  ready: function () {}
}
```
